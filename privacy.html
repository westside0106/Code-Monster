Datenschutzerklärung für die Code-Monster API

Stand: Juli 2025

1. Allgemeines

Die Code-Monster API ist eine produktionsreife Analyse- und Optimierungsplattform für Quellcode. Beim Zugriff auf die API werden ausschließlich technisch notwendige Daten verarbeitet, um die angefragten Funktionen korrekt bereitzustellen.

2. Verantwortlicher Anbieter

Philipp M. (alias Code-Monster)
E-Mail: code@monster.dev

3. Welche Daten werden verarbeitet?

  • Code-Snippets: Inhalte, die an Endpoints wie /analyze, /refactor etc. gesendet werden.
  • Tracebacks: Fehlerlogs an /explaintraceback.
  • Meta-Daten: Zeitpunkt, API-Route, technische Header.

4. Keine Speicherung oder Weitergabe

Alle Inhalte werden ausschließlich transient verarbeitet. Keine Speicherung, keine Weitergabe.

5. Zweck der Verarbeitung

Ausschließlich technische Analyse, Optimierung oder Transformation des Codes.

6. Rechtsgrundlage

Art. 6 Abs. 1 lit. b DSGVO (Vertragliche Erfüllung).

7. Betroffenenrechte

Da keine Daten gespeichert werden, entfallen klassische Rechte auf Löschung oder Auskunft.

8. Sicherheit

SSL-Verschlüsselung, Zugriffsschutz, API-Limiting.
