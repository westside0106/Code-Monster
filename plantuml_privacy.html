Datenschutzerklärung für die PlantUML API

Stand: Juli 2025

1. Allgemeines

Die PlantUML API ermöglicht die serverseitige Generierung von UML-Diagrammen (SVG) aus übermitteltem PlantUML-Text. Es erfolgt keine personenbezogene Verarbeitung.

2. Verarbeitete Daten

  • UML-Text: Der übermittelte PlantUML-Text wird verarbeitet, um ein Diagramm zu erzeugen. Dieser Text wird nicht gespeichert.
  • Verbindungsdaten: Technische Informationen wie IP-Adresse, Zeitstempel und HTTP-Header können serverseitig protokolliert werden.

3. Keine Speicherung des UML-Codes

Die Datenverarbeitung erfolgt ausschließlich im RAM für die Dauer der Anforderung. Keine Speicherung auf dem Server.

4. Zweck der Verarbeitung

Die Verarbeitung erfolgt ausschließlich zum Zweck der visuellen Darstellung des UML-Textes in Diagrammform.

5. Rechtsgrundlage

Art. 6 Abs. 1 lit. f DSGVO – berechtigtes Interesse an der Bereitstellung einer Diagramm-Vorschaufunktion.

6. Verantwortlicher Dienst

PlantUML Server
Web: https://plantuml.com
Betreiberinformationen siehe Impressum der offiziellen Seite.

7. Sicherheit

Alle Verbindungen erfolgen verschlüsselt (HTTPS). Keine Weitergabe der Daten an Dritte.

8. Kontakt

Bei Fragen zur Datenverarbeitung siehe PlantUML FAQ.
